# Elixir - EEX
## Root-me challenge - source code
### How to
Firstly, install `elixir` (challenge developed and tested with version `1.12.3`).
```bash
$> brew install elixir
...
$> apt-get install elixir
```
In the project root directory, fetch dependencies and launch interactive elixir:
```bash
$> mix deps.get
$> iex -S mix
```
Start hacking on http://localhost:4001/ !
